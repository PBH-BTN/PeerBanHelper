# Security Policy

## Supported Versions

Only latest release and dev version accept the Security report.  

## Reporting a Vulnerability

Please submit a security report via GitHub. We will respond to your security report within 7 days.  
If you do not receive a response as expected, it is possible your report was buried under other notifications.  
Please send an email to `ghostchu [at] qq.com` and CC `1683017633 [at] qq.com`.
